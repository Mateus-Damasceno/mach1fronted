Este é um teste de programação frontend

vamos fazer o layout de um cadastro de alunos basicos e uma listagem.

cada sessao tem uma particularidade. Nessa tela é possivel ver um titulo, corpo do texto e um botao que leva para a pagina de cadastro e ao lado desses texto está uma imagem

Foi utilizado:
  • Bootstrap: Para tornar o site responsivo e melhorar a aparencia da pagina
  • javascript: Para tratar a inserçao dos dados, respeitando as restriçoes de cada campo
Cadastro de Alunos

Dica 1

Site criado no estilo onepage utilizando html, Bootstrap e javascript para o desafio Mach1. Nessa pagina podemos ver um titulo para a sessao inteira e abaixo, o corpo de texto com uma imagem ao lado Para fazer o cadastro é necessario preencher todos os campos corretamente respeitando os filtros.
  • Nome: restricao de nº de caracteres
  • E-mail: deve conter @ no campo
  • CPF: cadastrar um numero de CPF válido
  • Curso: sem restrição
